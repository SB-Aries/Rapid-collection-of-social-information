# 网页邮箱与手机号爬取工具

一款基于 Python Tkinter 开发的可视化爬虫工具，支持单 URL / 批量 URL 爬取网页中的邮箱和中国手机号，全程自动去重，操作简单，结果可直接保存。

## 📌 功能特点
| 功能 | 说明 |
| :--- | :--- |
| **🖥️ 可视化 GUI** | 无需命令行操作，全图形界面交互，新手友好。 |
| **📝 双爬取模式** | 支持「单 URL 爬取」和「批量 URL 爬取（从 TXT 读取）」。 |
| **🧹 全流程去重** | 自动去重 URL 列表、单 URL 内数据、多 URL 间全局数据，最终仅保留唯一结果。 |
| **📱 手机号适配** | 支持提取多种格式的中国手机号，自动清洗为纯 11 位数字。 |
| **📧 邮箱提取** | 兼容主流邮箱格式，精准提取并去重。 |
| **💾 结果保存** | 爬取结果自动排序后保存为 TXT 文件。 |
| **🌐 网络检测** | 启动前自动检查网络连接，避免无效操作。 |
| **📦 自动装依赖** | 检测到缺失 `requests` 库时自动安装，无需手动配置。 |

## 🛠️ 环境要求
- **Python 版本**：Python 3.6 及以上（推荐 3.8+）
- **系统兼容**：Windows / Linux / macOS（全平台支持）
- **依赖库**：仅需 `requests`（代码会自动检测并安装）

## 🚀 使用步骤
1.  **运行代码**
    将代码保存为 `web_scraper.py`，直接运行：
    ```bash
    python web_scraper.py
    ```

2.  **选择爬取模式**
    - **模式1：单 URL 爬取**
        1.  选择「单 URL 爬取」单选框。
        2.  在「输入内容」框中填写完整 URL（必须带 `http://` 或 `https://`）。
        3.  勾选需要爬取的内容（邮箱 / 中国手机号，至少勾选一项）。
        4.  点击「开始爬取」。
    - **模式2：批量 URL 爬取**
        1.  准备一个 TXT 文件，每行填写一个完整 URL。
        2.  选择「批量 URL 爬取（从 TXT 读取）」单选框。
        3.  点击「选择 TXT 文件」，选中准备好的文件。
        4.  勾选需要爬取的内容，点击「开始爬取」。

3.  **结果操作**
    - **查看结果**：过程和结果实时显示在文本框中。
    - **手动去重（可选）**：点击「手动去重结果」进行强制兜底去重。
    - **保存结果**：点击「保存结果」，工具会自动创建文件夹并生成 `unique_emails.txt` 和 `unique_china_phones.txt`。
    - **清空结果**：点击「清空结果」重置所有数据和日志。

## ⚠️ 注意事项
- URL 必须填写完整（含 `http://` 或 `https://`）。
- 爬取前工具会**自动检测网络**。
- 本工具仅用于**学习和合法场景**，禁止爬取非公开或侵权数据。
- 批量 TXT 文件建议使用 UTF-8 编码，每行一个 URL，空行会被自动过滤。

## ❓ 常见问题
- **Q1：爬取失败？**
    - 检查 URL 是否可访问、网络是否正常。
    - 目标网站可能有反爬策略。
- **Q2：爬取结果为空？**
    - 目标网页本身可能无相关数据。
    - 网页内容可能是动态加载的（工具仅爬取静态 HTML）。
- **Q3：保存的文件为空？**
    - 爬取过程中未提取到有效数据。
    - 未勾选对应的爬取选项。
